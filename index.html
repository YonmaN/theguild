theguild by YonmaN

Israeli role players repository for profiling and finding other enthusiasts

The Guild is a repository of player profiles for role playing games. Find the perfect match for your group or join up with others like you!

Dependencies

Zend Framework 2 beta3 Moo Tools

Install

Extract the project Install ZF2 into vendors/ZendFramework
